# Android_projects

This repository contains three university Android project stages. Each new project is larger and more complex than the previous one

## Projects

### Project 1

`Proj_1` is the first Android assignment, a simple expense recording app that stores daily money records and shows statistics for weekly and monthly ranges

- Folder: [Proj_1](Proj_1)
- README: [Proj_1/README.md](Proj_1/README.md)
- Assignment brief: [Proj_1/doc/AndroidPLH2526_ergasia1.pdf](Proj_1/doc/AndroidPLH2526_ergasia1.pdf)

### Project 2

`Proj_2` is the second Android assignment, a Firebase-based messenger with authentication and a realtime database backend

- README: [Proj_2/README.md](Proj_2/README.md)

### Project 3

`Proj_3` is the final assignment and it is split into two separate Android apps:

- [UnipiCityVibe](Proj_3/UnipiCityVibe): an events app for nearby events, ticket reservations, notifications, and map-based discovery
- [UnipiAudioStories](Proj_3/UnipiAudioStories): a children’s story app with text-to-speech playback, story statistics, and multilingual UI

- Folder: [Proj_3](Proj_3)
- README: [Proj_3/README.md](Proj_3/README.md)
